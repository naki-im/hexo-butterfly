## 0.引言

这里是本站的隐私政策，本政策旨在助您理解我（们）对您的隐私数据等的收集和使用情况，本政策详细记录了双方的权责划分、具体使用细则等。

修订时间：2026年2月15日14:05（北京时间）

## 1. 我们收集的数据

我们收集的数据仅限于维持网站运行和提供基本功能所需。以下列表尽可能完整地列举了相关情况：

1. **服务商托管日志**：当您访问本站时，我们的托管服务商可能会自动生成并记录标准日志，数据可能会包括：您的IP地址、访问时间、请求的页面URL、浏览器类型和版本（User-Agent）。
2. **匿名化分析数据**：我们使用Umami云服务（[us.umami.is](https://us.umami.is)）进行网站流量统计。
3. **第三方账号信息（GitHub）**：当您使用本站的Giscus评论系统时，GitHub会根据其政策收集您的GitHub账号信息以标识您的身份。我们不直接收集或存储您的GitHub信息，也无法处理这些数据。

## 2. 您的数据的用途与共享

我们严格限制数据的使用范围，并明确区分我们与第三方的责任：

1. **服务商托管日志**：
   - **用途**：用于云服务商内部的运维保障、安全分析及满足特定地区的监管要求。
   - **共享**：这些日志存储于服务商的系统内，由托管服务商负责保密和处理。在接到司法机关合法、有效的命令时，其可能会依法提供此类日志以协助调查。
2. **匿名化分析数据（Umami）**：
   - **用途**：用于分析网站流量趋势，了解内容受欢迎程度，以改进本站。
   - **共享**：这些数据由Umami云服务处理。您可随时在[此公开仪表盘](https://us.umami.is/share/UYC5AaIszA6JxmGP/)查看公开的汇总结果。
3. **GitHub评论数据**：
   - **用途**：为本站提供评论功能。
   - **共享**：您通过Giscus提交的评论及相关GitHub个人信息，完全存储并受控于GitHub，受[GitHub隐私政策](https://docs.github.com/zh/site-policy/privacy-policies)约束。我们无法控制GitHub对您数据的处理。
4. **第三方工具调用**：
   - 当您访问本站时，为了显示访客 IP 所属地信息，您的浏览器会向第三方 API 发送请求，这可能使第三方获取到您的 IP 地址。我们无法控制该第三方对数据的处理。
## 3. 您的权利与控制

您对自己的数据拥有控制权：

- **关于评论**：您可以随时在GitHub上访问、修改或删除您发布的评论。
- **关于分析**：Umami在设计上已避免追踪个人。如果您希望进一步规避任何分析工具，可以考虑使用浏览器的“不跟踪”功能或安装内容拦截插件。
- **关于日志**：由于服务器日志是服务商内部运维所必需且通常已经过处理，一般情况下不提供个人访问和删除权，除非法律另有规定。
- **关于我们**：我们无法对存储并受控于第三方服务的数据负管理责任，也无法控制第三方服务商的数据留存政策和处理细节，如有疑惑，请查询其隐私政策。

>免责声明：我们仅在本政策中如实陈述网站与第三方工具的交互方式，不对第三方服务商的实际数据处理行为作出任何保证或承担任何责任。
